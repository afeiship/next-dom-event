# next-dom-event
> Dom event api based on next toolkit.

## installation
```bash
npm install -S @feizheng/next-dom-event
```

## apis
| api | params | description          |
| --- | ------ | -------------------- |
| on  | -      | register a new event |

## usage
```js
import NxDomEvent from '@feizheng/next-dom-event';
// code goes here:
```
